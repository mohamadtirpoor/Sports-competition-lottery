# 🏆 پلتفرم مدیریت مسابقات ورزشی

یک پلتفرم کامل فارسی برای مدیریت و قرعه‌کشی مسابقات ورزشی (پینگ‌پنگ، فوتبال‌دستی و...)

## ✨ ویژگی‌های اصلی

- ✅ ایجاد مسابقات حذفی و گروهی
- ✅ افزودن نامحدود شرکت‌کننده (بازیکن/تیم)
- ✅ قرعه‌کشی تصادفی با انیمیشن جذاب
- ✅ جدول درختی (Bracket) برای مسابقات حذفی
- ✅ جدول رده‌بندی برای مسابقات گروهی
- ✅ ثبت نتایج و به‌روزرسانی خودکار
- ✅ سیستم پیش‌بینی و رتبه‌بندی
- ✅ رابط کاربری فارسی و مدرن
- ✅ تم روشن، تاریک و ورزشی
- ✅ ریسپانسیو (موبایل، تبلت، دسکتاپ)

## 🛠 تکنولوژی‌ها

- **Frontend**: Next.js 14, React, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express
- **Database**: SQLite
- **Font**: Vazirmatn

## 📦 نصب و راه‌اندازی

### پیش‌نیازها

- Node.js (نسخه 16 یا بالاتر)
- npm یا yarn

### مراحل نصب

1. نصب وابستگی‌ها:
```bash
npm run install-all
```

2. اجرای پروژه در حالت توسعه:
```bash
npm run dev
```

این دستور هم بک‌اند و هم فرانت‌اند را به صورت همزمان اجرا می‌کند.

- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

### اجرای جداگانه

اگر می‌خواهید بک‌اند و فرانت‌اند را جداگانه اجرا کنید:

```bash
# بک‌اند
npm run dev:backend

# فرانت‌اند (در ترمینال دیگر)
npm run dev:frontend
```

## 🎯 نحوه استفاده

### 1. ایجاد مسابقه جدید
- روی دکمه "ایجاد مسابقه جدید" کلیک کنید
- نام مسابقه را وارد کنید
- نوع مسابقه (حذفی یا گروهی) را انتخاب کنید

### 2. افزودن شرکت‌کننده‌ها
- نام بازیکنان یا تیم‌ها را وارد کنید
- حداقل 2 شرکت‌کننده نیاز است

### 3. قرعه‌کشی
- روی دکمه "شروع قرعه‌کشی" کلیک کنید
- انیمیشن قرعه‌کشی نمایش داده می‌شود
- جدول مسابقات به صورت خودکار ساخته می‌شود

### 4. ثبت نتایج
- روی هر بازی کلیک کنید
- نتیجه را وارد کنید
- برنده را انتخاب کنید
- جدول به صورت خودکار به‌روزرسانی می‌شود

### 5. مشاهده رتبه‌بندی
- در تب "رتبه‌بندی" می‌توانید پیش‌بینی‌های کاربران را ببینید

## 🎨 تم‌ها

پلتفرم دارای 3 تم مختلف است:
- ☀️ **روشن**: برای استفاده در روز
- 🌙 **تاریک**: برای استفاده در شب
- ⚽ **ورزشی**: تم ویژه ورزشی

## 🔌 API Endpoints

### مسابقات
- `POST /api/tournaments` - ایجاد مسابقه جدید
- `GET /api/tournaments` - دریافت لیست مسابقات
- `GET /api/tournaments/:id` - دریافت جزئیات مسابقه

### شرکت‌کننده‌ها
- `POST /api/tournaments/:id/participants` - افزودن شرکت‌کننده

### قرعه‌کشی
- `POST /api/tournaments/:id/draw` - انجام قرعه‌کشی

### نتایج
- `PUT /api/matches/:id/result` - ثبت نتیجه بازی

### پیش‌بینی
- `POST /api/predictions` - ثبت پیش‌بینی
- `GET /api/tournaments/:id/leaderboard` - دریافت رتبه‌بندی

## 🚀 دیپلوی در لیارا

### پاسخ سوال شما:
پروژه شامل **دو بخش** است:
- **Backend**: Node.js (Express) - پورت 5000
- **Frontend**: Next.js - پورت 3000

### روش توصیه شده: دو پروژه جداگانه

#### 1. Backend (Node.js)
```bash
cd backend
liara deploy --platform=node --port=5000
```
پلتفرم: **Node.js**

#### 2. Frontend (Next.js)
```bash
cd frontend
liara deploy --platform=next
```
پلتفرم: **Next.js**

برای جزئیات بیشتر، فایل `DEPLOY.md` را مطالعه کنید.

### تنظیمات مهم:

1. در پروژه Backend لیارا:
   - پلتفرم: Node.js
   - پورت: 5000

2. در پروژه Frontend لیارا:
   - پلتفرم: Next.js
   - متغیر محیطی: `NEXT_PUBLIC_API_URL=https://[backend-url].liara.run/api`

3. در Backend، CORS را برای دامنه Frontend تنظیم کنید
